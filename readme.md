# Senator-Code-Challenge

Small Kotlin based Android app to take the result of [this](https://www.govtrack.us/api/v2/role?current=true&role_type=senator) site and display it in a Recycler View

---

## Libraries Used ##

- Retrofit2
- RecyclerView
- Kotlin Coroutines
